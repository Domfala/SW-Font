This repository contains the official implementation of our research: **SW-Font**. 
We propose a novel **Geometry-Guided Content Adaptation (G-GCA)** module and a **Width Estimation Network (WES-Net)** to solve the problem of structural inconsistency and stroke width instability in few-shot font generation. By utilizing physically-grounded stroke force information, our method achieves superior performance in maintaining style consistency.
